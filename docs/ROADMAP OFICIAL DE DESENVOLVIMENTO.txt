ROADMAP OFICIAL DE DESENVOLVIMENTO — sisRUA

Visão: Pipeline técnico Campo → GIS → CAD, offline-first, custo zero inicial, preparado para nuvem sem depender dela.

PRINCÍPIOS ARQUITETURAIS (IMUTÁVEIS)

Offline-first (campo e projeto)

CRS tratado no início do fluxo

Frontend: EPSG:4326 (Lat/Long)

Backend/CAD: SIRGAS 2000 / UTM (zona automática)

Formato aberto e padrão: GeoJSON enriquecido

Separação clara de responsabilidades

JS → visualização

C# → orquestração AutoCAD

Python → GIS + conversão CAD

Nenhuma dependência de nuvem até provar valor

FASE 1 — MVP: PLUGIN DE DESENHO DE RUAS
Objetivo

Permitir a criação manual e importada de ruas com medidas corretas no CAD.

Funcionalidades

Preview no mapa

Backend (Python)

Conversão imediata para UTM

Layers corretos

FASE 1.5.1 — PERSISTÊNCIA DE DADOS COM SQLITE
Objetivo

Permitir salvar e recuperar projetos desenhados para redesenho rápido.

Entregas

Integração SQLite local (projects.db)

Comando `SISRUA_SAVE_PROJECT` (salvar projeto atual)

Comando `SISRUA_RELOAD_PROJECT` (listar e redesenhar projetos salvos)

Benefício

Recuperação de trabalho

Reutilização de projetos

Base para auditoria/gestão de histórico

FASE 1.5.2 — LIMPEZA E SIMPLIFICAÇÃO DE GEOMETRIA OSM
Objetivo

Melhorar a qualidade dos desenhos CAD gerados a partir de OSM, reduzindo linhas duplicadas, sobrepostas ou complexas.

Entregas

Remoção de Polylines Duplicadas Exatas

Fusão de Polylines Colineares/Contíguas

Simplificação de Polylines (Douglas-Peucker)

Benefício

Desenhos mais limpos e performáticos

Redução de erros em operações CAD subsequentes

Experiência de usuário aprimorada

FASE 1.5.7 — IMPORTAÇÃO DE EDIFICAÇÕES DO OSM
Objetivo

Adicionar a capacidade de importar e desenhar os contornos das edificações adjacentes à malha viária a partir do OpenStreetMap.

Entregas

Opção na UI para "Incluir Edificações"

Layer dedicada no AutoCAD (SISRUA_OSM_EDIFICIOS)

FASE 1.5.8 — DESENHO DE BLOCOS A PARTIR DE PONTOS INSERIDOS NA UI
Objetivo

Converter marcadores inseridos manualmente pelo usuário no mapa da interface em blocos do AutoCAD.

Entregas

Coleta de marcadores da UI (Postes, Árvores)

Botão "Desenhar Pontos no CAD"

Envio de GeoJSON para o plugin C#

Inserção de blocos nas coordenadas corretas

FASE 1.6 — APERFEIÇOAMENTO DA EXPERIÊNCIA DE INSTALAÇÃO E PRIMEIRO USO (UX)
